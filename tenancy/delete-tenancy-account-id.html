DELETE /tenancy/account/{accountId}

Deletes the specified account.
Token usage: BlueXP user token.

Parameters

Name Type In Required Description

accountId

string

path

True

authorization

string

header

True

X-User-Token

string

header

False

X-Service-Request-Id

string

header

False

Status: 200, Default response
