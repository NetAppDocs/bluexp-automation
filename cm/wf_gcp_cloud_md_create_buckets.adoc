= Create Buckets
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./media/

[.lead]
You can perform this workflow to create a new bucket. This is a GCP storage bucket (data container in GCP) required for data tiering and backup.

Choose the correct workflow depending on the type of Cloud Volumes ONTAP deployment:
* <<Create working environment for single node, Single Node>>
* <<Create working environment for high availability pair, HA pair>>

== Create working environment for single node
You can use this workflow to create single node working environment with BYOL licensing.
=== 1. Create a bucket

[cols="25,75"*,options="header"]
|===
|HTTP method
|Path
|POST
|/occm/api/gcp/vsa/metadata/create-bucket
|===

curl example::
[source,curl]
curl --location --request POST 'https://cloudmanager.cloud.netapp.com/occm/api/gcp/vsa/metadata/create-bucket' --header 'x-agent-id: <AGENT_ID>' --header 'Authorization: Bearer <ACCESS_TOKEN>' --header 'Content-Type: application/json' --d JSONinput


JSON input example::
[source,json]
{
  "projectId": "occm-dev",
  "bucketName": "zivgcpbucket02",
  "location": "us-west1",
  "storageClass": "standard"
}

Output::

None
